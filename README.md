# AI Job Market Analysis and Salary Prediction Using Machine Learning

## 🔹 Objective
To analyze AI job market data and predict industry salaries by identifying key factors such as experience level, job role, company size, and other job-related features using machine learning techniques.

## 🔹 Tech Stack
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn  
Jupyter Notebook / Google Colab  

## 🔹 Key Steps
Data Cleaning and Preprocessing  
Exploratory Data Analysis (EDA)  
Data Visualization  
Feature Engineering  
Machine Learning Model Building  
Model Evaluation using MAE, RMSE, and R² Score  

## 🔹 Key Insights
Analyzed salary trends across different AI job roles and experience levels  
Identified important factors affecting AI industry salaries  
Visualized salary distribution and job market patterns  
Built a machine learning model to predict expected salaries based on job-related features  

## 🔹 Conclusion
This project demonstrates how data analysis and machine learning techniques can be used to understand AI job market trends and predict salary patterns, helping in better career and workforce insights.
